Development Tools

Visual Studio

Visual Studio Code is a free source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, snippets, code refactoring, and embedded Git.

GitHub

GitHub, Inc. is a subsidiary of Microsoft which provides hosting for software development and version control using Git. It offers the distributed version control and source code management (SCM) functionality of Git, plus its own features.

Android Studio

Android Studio is the official IDE for Android app development. It provides tools for coding, testing, debugging, and building Android applications efficiently using Java and Kotlin.

Kaggle Platform

Kaggle is a community for data scientists and machine learning engineers. It allows users to share datasets, participate in competitions, and build predictive models in a collaborative environment.

Google colab

Google Colab is a cloud-based Jupyter Notebook service that allows you to write and execute Python code for data analysis and machine learning using free GPU and TPU resources.

Unity Engine

Unity is a powerful cross-platform engine for creating interactive 2D, 3D, VR, and AR experiences. It is widely used for developing games, simulations, and immersive visual applications.

Arduino IDE

Arduino IDE is an open-source platform used for programming microcontrollers. It simplifies prototyping hardware-based projects for IoT, robotics, and embedded systems.

Postman (API)

Postman is an API testing and development tool that simplifies sending requests, automating workflows, and documenting APIs for backend and web service development.
